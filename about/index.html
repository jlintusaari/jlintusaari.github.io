Regarding machine learning I am interested in using it as a tool for expanding the set of problems that can be automated. Automation can mean e.g. helping an online shopper find relevant products faster via optimized search or just solving a Boggle game from an image. Automation can also mean software that is efficient for its purpose. I am one of the core developers of the open-source Python software called ELFI which comprises a set of machine learning methods for simulator-based inference and was published in JMLR MLOSS. ELFI provides a convenient interface for specifying simulator-based inference problems using graphs, and streamlines many tasks in performing the inference, including parallelization to a computational cluster.
